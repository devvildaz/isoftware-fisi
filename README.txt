PROCEDIMIENTO PARA EJECUTAR:

Importante: Estar ubicado(a) en la carpeta isoftware-fisi\

1.Instalar dependencias
	1.1 Abrir una consola
	1.2 Ejecutar el siguiente comando:
		pip install -r .\Entregable-01\requirements.txt

2. Cargar las imágenes
	2.1 Cargar las imágenes (manualmente) que desea analizar a la ruta Entregable-01\images\
	*Nota: Ya existen imágenes de prueba

3. Ejecutar el programa principal
	3.1 Ejecutar el siguiente comando:
		python Entregable-01\ui\src\app.py

4. Modo de uso
	4.1 Hacer click en el botón ubicado en la esquina superior derecha
	4.2 En el explorador navegar hacia isoftware-fisi\Entregable-01\ 
	4.3 Seleccionar images\ y dar click en aceptar
	4.4 Seleccionar una imagen y ver resultados
